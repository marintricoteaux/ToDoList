# ToDoList

A simple **To-Do List web application** built with **HTML, CSS and JavaScript**.

This project is part of my learning process in web development. The goal is to progressively improve the application by adding features and refactoring the code while learning new concepts.

## Features

* Add a new task
* Edit an existing task
* Delete a task
* Validate tasks with **Enter** or buttons
* Preserve the checkbox state when editing a task

## Tech Stack

* HTML
* CSS
* JavaScript (Vanilla)

## Project Goals

The objective of this project is not only to build a working To-Do List, but also to progressively improve the code structure and implement more advanced front-end concepts.
